# How to run
get your anime list export at https://myanimelist.net/panel.php?go=export <br>
in completionist-cli.py add the path to: <br>
MAL_EXPORT_PATH = "YOUR_PATH"
<br>
<br>
<br>
python3 -m venv .venv<br>
source .venv/bin/activate<br>
pip install -r requirements.txt<br>
then run:<br>
python3 completionist-cli.py --help
<br>
